# robosys_2020_kadai2

# 概要
ROSを用いて1から順にカウントし、3の倍数でnasu、15の倍数でnasubiと出力するものです。  
ロボットシステム学第10回で使ったものに手を加えました。  


# 動作環境
Raspberry Pi 4 Model B
ubuntu20.04  
# 実行方法 
1つ目の端末 roscore  
2つ目の端末 chmod +x count.py  
2つ目の端末 chmod +x nasu.py  
2つ目の端末 roslaunch mypkg mypkg.launch
